# Quick Start Guide - Advanced Features

## Getting Started with New Features

### 1. Product Reviews

**How to Leave a Review:**
1. Navigate to any product page
2. Click "Write a Review" button (requires login)
3. Select 1-5 star rating
4. Enter review title (5+ characters)
5. Add optional comment (max 2000 characters)
6. Click "Submit Review"

**Viewing Reviews:**
- Reviews appear below product description
- Sorted by newest first
- Shows rating, verified purchase badge, and helpful count
- Click "Helpful" button to mark review useful

---

### 2. Wishlist Management

**Add to Wishlist:**
1. Click heart icon on product page or listing
2. Product saves to your wishlist
3. Visit `/wishlist` to view all saved items

**Wishlist Actions:**
- View all wishlist items with prices
- Remove items individually
- Clear entire wishlist
- See current product prices
- View product ratings

**Quick Access:**
- Link in navigation menu
- Heart icon on product pages

---

### 3. Advanced Search & Filtering

**Access Search:**
- Click Search icon in navbar
- Navigate to `/search` directly

**Filter Options:**
- **Search Term**: Search product names and descriptions
- **Category**: Filter by Electronics, Home & Living, Fashion, Sports
- **Price Range**: Use sliders for min/max price
- **Rating**: Show only products with minimum rating
- **Sort By**: Newest, Price (low-high), Price (high-low), Rating, Popular

**Tips:**
- Combine multiple filters for specific results
- Use price range to find products in your budget
- Sort by rating to find top-reviewed items
- Search updates in real-time

---

### 4. Coupon & Discounts

**Apply Coupon at Checkout:**
1. Proceed to checkout
2. Enter coupon code in discount field
3. Click "Apply Coupon"
4. Discount automatically calculated
5. See new total amount

**Sample Coupons:**
- SUMMER20 (20% off summer items)
- WELCOME10 (10% off first purchase)
- FREE50 ($50 off orders over $200)

**Coupon Rules:**
- One coupon per order
- Minimum purchase amount may apply
- Limited usage per user
- Valid only during specified date range

---

### 5. Return Management

**Create Return Request:**
1. Go to Orders page
2. Find order containing item to return
3. Click "Request Return"
4. Enter order ID, product ID, quantity
5. Select return reason from dropdown
6. Add optional details/explanation
7. Submit request

**Return Status Tracking:**
- View all return requests on `/returns`
- Check current status: Pending → Approved → Refunded
- See estimated refund amount
- View request date and processing date

**Valid Return Reasons:**
- Defective/Broken
- Not as described
- Size/Fit issue
- Changed mind
- Found better price
- Other

**Return Timeline:**
- **Pending**: Initial status, awaiting admin review
- **Approved**: Admin approved, refund being processed
- **Rejected**: Admin declined return request
- **Refunded**: Refund completed to original payment method

## Common Workflows

### Complete Shopping Experience:

```
1. Browse Products
   └─ Use search/filters or browse categories

2. Read Reviews
   └─ Check product ratings and customer feedback

3. Add to Wishlist (Optional)
   └─ Save items for later

4. Add to Cart
   └─ Click "Add to Cart" button

5. Review Cart
   └─ Check quantities and prices

6. Apply Coupon (Optional)
   └─ Enter discount code if available

7. Checkout
   └─ Enter shipping & payment info

8. Order Confirmation
   └─ Receive order number

9. Track Order
   └─ View status on Orders page

10. Leave Review (After Delivery)
    └─ Share your experience
```

---

### Return Product Process:

```
1. Receive Order
   └─ Verify contents and condition

2. Decide to Return
   └─ If defective or not satisfied

3. Request Return (on /returns page)
   └─ Fill return form with details

4. Submit Return
   └─ System sends to admin

5. Wait for Admin Review
   └─ Status: Pending

6. Admin Decision
   └─ Status: Approved or Rejected

7. If Approved: Receive Refund
   └─ Status: Refunded

8. Verify Refund
   └─ Check bank account/payment method
```

---

### Wishlist to Purchase:

```
1. Add Items to Wishlist
   └─ Click heart on product pages

2. View Wishlist
   └─ Navigate to /wishlist

3. Monitor Prices
   └─ Prices update regularly

4. Add to Cart
   └─ When ready to purchase

5. Checkout
   └─ Complete purchase process

6. Track Order
   └─ View on Orders page
```

## Troubleshooting

### Coupon Not Working?
- ✓ Check expiry date
- ✓ Verify minimum purchase amount
- ✓ Ensure you haven't exceeded per-user limit
- ✓ Confirm code spelling (case-insensitive)

### Can't Add to Wishlist?
- ✓ Log in to your account first
- ✓ Check that product is in stock
- ✓ Refresh page and try again

### Return Request Not Submitting?
- ✓ Verify order exists
- ✓ Check product is in that order
- ✓ Ensure quantity is correct
- ✓ Select a valid return reason

### Reviews Not Showing?
- ✓ Refresh page
- ✓ Check if review was approved by admin
- ✓ Verify you're looking at correct product
